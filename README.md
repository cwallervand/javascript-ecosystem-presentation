javascript-ecosystem-presentation
=================================

## 1-index.html
* Run `python -m SimpleHTTPServer 8000`
* Open http://localhost:8000/index.html

## 2-npm
* Run `npm install`
* Run `python -m SimpleHTTPServer 8000`
* Open http://localhost:8000/index.html

## 3-webpack-bundling
* Run `npm install`
* Run `python -m SimpleHTTPServer 8000`
* Open http://localhost:8000/index.html

## 4-webpack-devserver
* Run `npm install`
* Run `npm start` (no longer need for python server)
* Open http://localhost:8000/
